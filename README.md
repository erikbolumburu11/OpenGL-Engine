# OpenGL-Engine
[Video](https://youtu.be/lygBcTKR5fo?si=EZV7-a6U0Am7yHdJ)

Programmed a 3D renderer using the ‘Blinn-Phong’ reflection model to replicate realistic lighting using OpenGL.

### Libraries/Tools Used
- GLFW & GLAD
- CMake
- Git
- vcpkg
